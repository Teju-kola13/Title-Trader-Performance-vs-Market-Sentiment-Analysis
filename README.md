# Trader Performance vs Market Sentiment Analysis

## Overview

This project analyzes the relationship between Bitcoin market sentiment and trader performance using the Fear & Greed Index and Hyperliquid historical trading data.

## Objective

The goal is to investigate how different market sentiment conditions (Fear, Extreme Fear, Greed, and Extreme Greed) influence trading performance, profitability, win rates, and trading volume.

### 1. Fear & Greed Index Dataset

* Date
* Classification

### 2. Hyperliquid Historical Trading Dataset

* Account

* Size
* Side
* Timestamp
* Closed PnL
* Leverage
